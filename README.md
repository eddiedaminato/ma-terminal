# **Installation**

0. **Download: (ma_terminal.py & requirements-ma_terminal.py)**

1. **Install dependencies**
```bash
Windorws + R = CMD
pip install -r requirements-ma_terminal.txt
```
2. **Run the application**
```bash
python ma_terminal.py

```
